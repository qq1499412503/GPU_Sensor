# GPU_Sensor
sensor for detail information from gpu
current version 0.1
current support one GPU for V0.1 only

![image](https://github.com/lexsaints/powershell/blob/master/IMG/ps2.png)

# how to use
run gpu.py with following package: 
pyqt5, pynvml

pip install pyqt5
pip install pynvml
